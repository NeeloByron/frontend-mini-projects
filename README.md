## 🚀 Projects

### 01 — Todo CLI App
**The foundation.** A command-line task manager that demonstrates core Node.js,
OOP with ES6 classes, file system persistence, and clean argument parsing.

**Stack:** `Node.js` `Commander.js` `fs module` `OOP`

📁 [View project](./01-todo-cli/) · ✅ Complete

---

### 02 — REST API — Task Manager
**The backend.** A full CRUD REST API with user authentication, PostgreSQL database,
and tested endpoints. The kind of API you'd actually ship.

**Stack:** `Node.js` `Express.js` `PostgreSQL` `JWT` `bcrypt` `Jest` `Supertest`

📁 [View project](./02-task-manager-api/) · ✅ Complete

---

### 03 — Fullstack Budget Tracker
**The full picture.** A web application with an Express backend, vanilla JS frontend,
and PostgreSQL database. Track income, expenses, and view summaries by category.

**Stack:** `Node.js` `Express.js` `PostgreSQL` `HTML` `CSS` `Vanilla JS` `Chart.js`

📁 [View project](./03-budget-tracker/) · 🔄 In Progress

---

### 04 — Cloud Notes App *(Azure + Node.js)*
**The differentiator.** A notes application that stores data in Azure Blob Storage
instead of a local database — combining fullstack JS skills with Azure cloud.

**Stack:** `Node.js` `Express.js` `Azure Blob Storage SDK` `HTML` `CSS`

📁 [View project](./04-cloud-notes-app/) · 🔄 In Progress
